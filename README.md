В рамках проекта была поставлена цель протестировать API веб-приложения "Яндекс. Прилавок".

Использованные инструменты: Postman, apiDoc.

Применённые виды тестирования: функциональное тестирование, API-тестирование.

Применённые техники тест-дизайна: классы эквивалентности, граничные значения.

Выполнено:

1. Подготовлен чек-лист из 111 запросов, выполнено тестирование, обнаружено 18 багов.

2. Составлены 18 баг-репортов.
